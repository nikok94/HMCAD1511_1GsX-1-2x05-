
Исходные директории:

edk - файлы для дизайна в XPS; Из исходных только *.xmp и *.mhs;
sim - файлы, задействованные только симуляции;
src - основные исходники;
tcl - скрипты для работы с проектом;
ucf - исходные файлы ограничений.

Производные директории:
prj - собранный проект для PlanAhead.

Сборка проекта:

 запустить build_win дождаться конца выполнения процесса сборки


После окончания сборки все временные файлы проекта находятся в папке prj
